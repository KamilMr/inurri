The admin panel was created as an internal part of ProsteZapisy. It lets the user manage the calendar, availability, and booking settings from one place. The user can prepare available times in the panel, and clients later see them through the public booking link.

### Scope of work
* Admin panel
* Calendar view
* Availability management
* Time block management
* User settings
