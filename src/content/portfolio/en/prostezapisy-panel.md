## Project goal

The admin panel was created as part of the ProsteZapisy system. Its role is to let users manage their calendar, availability, and settings without using a large or complicated tool.

This is a private part of the application, so it does not link to a public website or a visitor-facing view. Users access it after signing in to prepare time slots that clients later see through the public booking link.

## Scope

- admin panel implementation in React,
- calendar view with a weekly schedule layout,
- sidebar navigation with a mini calendar,
- availability and time-block management,
- user settings section,
- integration with the same backend that powers public bookings.

## Approach

The panel was designed so that the most important part of the system — the calendar — is visible immediately after entering the application. The sidebar provides quick navigation, while the main screen focuses on availability and scheduled time blocks.

React was used to build the interactive panel interface. Data, availability, and settings are handled by the same backend used by the public ProsteZapisy booking flow.

## Result

The result is a private management panel that complements the product website and public booking view. Together, they make ProsteZapisy a complete application: the user manages availability in the panel, while the client books through a simple public link.
